=== WooCommerce URL Coupons ===
Author: skyverge
Tags: woocommerce
Requires at least: 5.6
Tested up to: 6.1.1
Requires PHP: 7.4

Create a unique url that applies a discount and optionally adds one or more products to the customer's cart.

See https://docs.woocommerce.com/document/url-coupons/ for full documentation.

== Installation ==

1. Upload the entire 'woocommerce-url-coupons' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
