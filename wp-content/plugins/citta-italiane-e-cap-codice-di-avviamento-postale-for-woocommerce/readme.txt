Selettore delle città italiane in fase di checkout

== Description ==

Il plugin permette di selezionare le città italiane anzichè far trascrivere al cliente la città nel campo libero. 
Il plugin automaticamente compila il campo cap con la città selezionata (tranne per le 41 città che hanno più cap)
Possibilità di disattivazione dell'autocompilazione nelle impostazioni del plugin.


Con la versione [pro](https://4wp.it/product/citta-italiane-e-cap-automatico-per-woocommerce/) avrai una versione completa con:


- La possibilità di scelta del c.a.p per le 41 città aventi più c.a.p (Bologna,Bari,Catania,Milano,Roma,Messina,Napoli,Palermo,Firenze,Pisa,
Verbania,Trento,Alessandria,Ancona,Bergamo,Brescia,Cagliari,Ferrara,Foggia,Cesena,Forli,Genova,La Spezia,Livorno,Modena,
Padova,Parma,Perugia,Pesaro,Pescara,Piacenza,Ravenna,Reggio Calabria,Reggio Emilia,Rimini,Salerno,Taranto,Trieste,Venezia,
Verona,Torino)

- Impostazione per bloccare il campo cap, non permettendo la modifica (ideale per regole personalizzate di spedizione, con una migliore condizione di verifica tra città e cap).



Acquista la versione [pro](https://4wp.it/product/citta-italiane-e-cap-automatico-per-woocommerce/) da [qui](https://4wp.it/product/citta-italiane-e-cap-automatico-per-woocommerce/)
